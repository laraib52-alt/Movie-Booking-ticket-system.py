## Description

This project is a simple Python program that simulates a movie ticket booking system. It checks user eligibility based on age and show time, applies membership discounts, calculates extra charges for weekends or evening shows, and determines the final ticket price according to seat type.

## Features

* Age-based eligibility check
* Evening show restrictions
* Membership discount system
* Weekend and timing extra charges
* Seat type service charges
* Final price calculation

## Technologies Used

* Python (Basic Conditional Statements)

## How to Run

1. Open the program in any Python editor.
2. Run the file.
3. Enter the required details when prompted.
4. The system will display eligibility and final ticket price.
